# 博客恐怖 ARG 创作方法论

这份提炼不是复刻某个现成 ARG，而是学习它背后的创作技术：怎样让一个普通网页逐渐变成一场调查、一次窥探、一种仪式，最后让玩家意识到自己已经参与了故事。

## 一、最值得学习的不是设定，而是结构

可学习的是这些“功能结构”：

| 功能 | 作用 | 可替换形式 |
| --- | --- | --- |
| 现实入口 | 给玩家一个开始调查的理由 | 论坛帖、社媒寻人、旧链接、网盘、招聘页、校友录、商品评论 |
| 档案层 | 让玩家回看过去 | 假 Wayback、旧博客目录、相册备份、论坛历史帖、邮箱归档 |
| 私密层 | 让玩家越界进入角色内心 | 密码日记、好友信件、草稿箱、仅好友可见相册、隐藏留言 |
| 证据层 | 用“冷冰冰的系统”重写前面的情绪 | 学校记录、医院记录、物业公告、图书馆借阅、社团名单、车站时刻 |
| 异常层 | 网页开始不再只是网页 | UI 变形、文字污染、音频/图片异常、时间错位、页面回应玩家 |
| 后果层 | 玩家行为产生意义 | 保存、释放、拒绝、修复、删除、见证、误助某种东西 |

不要照搬的是具体人物、病历、死亡方式、复活主题、感官路线、域名、密码、目录名、特殊符号和结局措辞。

## 二、情绪递进公式

好的博客恐怖 ARG 通常不是“先吓人”，而是这样推进：

1. 可信  
   页面像真的旧网页，信息像真的生活碎片。

2. 亲近  
   玩家通过日记、相册、留言、朋友互动认识角色。

3. 不安  
   小矛盾出现：日期不对、说法冲突、照片解释不了、留言语气奇怪。

4. 证据化  
   外部系统出现，把私人叙述变成可查证的事实，玩家开始像侦探。

5. 污染  
   语言、颜色、排版、文件名、音频、导航逐步坏掉。

6. 卷入  
   页面不再只是被阅读，而是开始向玩家请求、指认、感谢或威胁。

## 三、谜题设计原则

谜题答案应该像“生活细节”，不是像“作者随便塞的密码”。

| 原则 | 做法 |
| --- | --- |
| 线索与锁分离 | A 页面给日期，B 页面要输入日期 |
| 重复三次 | 第一次像装饰，第二次像巧合，第三次才成为答案 |
| 答案有情绪 | 生日、旧昵称、房间号、喜欢的歌、约定地点、最后一张照片 |
| 错误也有反馈 | 错误提示可以暴露角色语气或世界观 |
| 不只靠源码 | 除非提前教过玩家“看源码是玩法” |
| 音频要有替代 | 声道、摩斯、倒放类谜题应有备用线索 |

## 四、氛围制作方法

不要简单堆恐怖词。让网页自己“变坏”。

| 维度 | 前期 | 中期 | 后期 |
| --- | --- | --- | --- |
| 色彩 | 可爱、清爽、年代感 | 灰、旧、褪色 | 暗红、黑、刺眼高对比 |
| 文字 | 日常、幼稚、碎碎念 | 断句、重复、错字 | 乱码、罕见字、命令式语气 |
| UI | 普通博客/论坛 | 失效链接、错位按钮 | 导航变成仪式词、页面主动改写 |
| 图片 | 相册、头像、贴纸 | 证据、局部、截图 | 遮挡、损坏、重复出现 |
| 音频 | 背景音乐 | 噪声、声道提示 | 语音、倒放、节奏谜题 |
| 时间 | 明确年月日 | 互相矛盾 | 不可能的今天、未来、循环 |

## 五、原创替换法

做类似作品时，可以把参考作品拆成“功能”，再换成自己的内容。

| 参考中的功能 | 不要照搬 | 可以替换成 |
| --- | --- | --- |
| 找失联网友 | 具体网友和博客域名 | 找旧游戏存档、校刊作者、匿名电台主播、失踪摄影账号 |
| 旧博客时间线 | 同样的年份和家庭悲剧 | 论坛账号成长史、相册备份、游戏公会记录、广播节目归档 |
| 医院/病历系统 | 同样的病历和科室 | 学校处分记录、图书馆逾期记录、物业维修单、社团入会表 |
| 生日密码 | 同样的生日数字 | 昵称、座位号、借书证号、歌词页码、车票时间 |
| 感官分支 | 同样的眼/耳/口路线 | 四季、房间、课程、颜色、梦境层级、旧设备按钮 |
| 复活诉求 | 同样的复活主题 | 道歉、删除、找回、证明存在、停止循环、归还名字 |

## 六、最小制作大纲

1. 玩家为什么会点进来？
2. 玩家一开始以为自己在找什么？
3. 角色真正想要什么？
4. 哪个网页承载“怀旧感”？
5. 哪个网页承载“私人越界”？
6. 哪个网页承载“冷证据”？
7. 哪个网页第一次主动回应玩家？
8. 玩家最后做了什么？
9. 这个行为是拯救、误救、见证，还是放手？

## 七、创作检查表

- 是否只借用了结构，而不是借用人物和独特设定？
- 每个谜题能否从玩家可见信息推出？
- 每个页面是否同时承担“剧情、线索、氛围”至少一种功能？
- 恐怖是否从生活细节里长出来，而不是靠突兀吓人？
- 分支是否最终回到主线？
- 结局是否回答了“玩家为什么重要”？
