Card Writer

缃笺

统一编辑人设、世界书、预设、记忆，并在同一工作台里直接查看预览。

就绪 0 个块 编辑时自动保存草稿，保存按钮用于落盘工程文件。

导出时会自动使用这个名字，例如「莉莉丝的人设.json」。

轮椅模式

配置当前 mod 专用的 API / 模型 / 提示词，不影响别的页面。

Persona

人设卡

固定编辑单 persona 的完整角色卡结构。

Worldbook

世界书

左边选词条，右边编辑全部字段；上方管理全局 settings。

Preset

预设

完整编辑 active preset、modules、extra prompts 与 prompt groups。

Memory

记忆

每条记忆独立编辑，支持标签、备注与正文。

Preview

预览

这里会显示当前人设卡编译结果，以及世界书 / 预设 / 记忆的实时 JSON。

终检警告

刷新预览时会同步校验；点警告可跳回对应位置修正。

未检查
点击「刷新预览」生成校验结果。

人设卡预览

点击「刷新预览」生成最终角色卡。
