# 心笺使用说明（v1.0.0）

心笺是 Fantareal 的结构化状态记录与幕笺展示工具。

主模型负责正文，心笺负责正文之外的状态记录与展示。

## 推荐组合

- 新手推荐：标准状态模板 + 标准样式
- 古风剧情：标准状态模板 + 纸页时间卡
- 演出分镜：标准状态模板 + 剧本分镜
- 状态面板：标准状态模板 + 状态面板 Pro

## 模板与美化包边界

模板决定输出什么内容；美化包决定这些内容怎么显示。

美化包不能删字段，也不能替代模板字段。自定义字段会继续完整显示。

## 数值系统

标准状态模板包含好感、信任、戒备、心绪四组通用数值。

格式统一为：

```text
65/100（+2）
80/100（+0）
15/100（-1）
```

其中 +0 表示本轮已判断但无明显变化。

## 表册说明

- 角色状态表：展示角色自身状态，并同步心绪等角色状态数值摘要。
- 关系表：展示关系摘要，并同步好感、信任、戒备等关系数值摘要。
- 数值变化记录：记录每回合 old → new → delta，用于测试重 roll、编辑重算和数值变化追踪。

## 重 roll 与编辑重算

心笺会根据 turn_id 记录本轮数值变化。重 roll 或编辑重算时，应撤回旧变化并重新应用新结果，避免重复叠加。
