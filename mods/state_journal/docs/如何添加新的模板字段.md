# 如何添加新的模板字段

本文面向心笺用户，说明如何在“幕笺模板”中添加新的角色状态字段。

## 1. 模板字段是什么

模板字段决定辅助模型需要生成哪些内容。字段属于“内容规则”，不是外观规则。

例如：

```text
情绪 emotion
衣着 clothing
角色神态 posture
角色互动 interaction
```

美化包只负责这些内容如何显示，不能删除字段。

## 2. 新字段需要填什么

在心笺设置页进入：

```text
幕笺模板 → 字段列表 → 新增字段
```

每个字段建议填写四项：

```text
字段 Key：英文或下划线，例如 visual_focus、field_11
显示名：中文名称，例如 视线焦点
生成说明：告诉辅助模型这个字段要写什么
输出模板：决定字段如何进入幕笺文本
```

## 3. 字段 Key 怎么写

推荐写法：

```text
visual_focus
body_motion
micro_reaction
field_11
```

不要频繁改字段 Key。字段 Key 一旦改了，旧模板输出和前端显示可能对不上。

## 4. 生成说明怎么写

生成说明要告诉辅助模型“写什么”和“不要写什么”。

示例：

```text
读取本轮正文，概括角色视线落点、注意力偏移和微表情。没有明确依据时保持含蓄，不要凭空新增重大动作。
```

## 5. 输出模板怎么写

输出模板使用 `{字段Key}` 占位。

示例：

```text
<情绪({emotion})>
<衣着({clothing})>
<视线焦点({visual_focus})>
```

新增字段后，也要把它写进输出模板：

```text
<新字段显示名({your_field_key})>
```

## 6. 数值字段怎么写

心笺标准数值字段推荐格式：

```text
favor_level   好感
trust_level   信任
guard_level   戒备
pulse_level   心绪
```

这些字段必须输出为：

```text
65/100（+2）
72/100（+0）
18/100（-1）
```

无变化也写 `+0`，不要只写 `0`。

## 7. 常见错误

### 字段不显示

检查：

```text
字段 Key 是否写进输出模板
辅助模型是否真的输出该字段
美化包是否为新版“不裁剪字段”逻辑
```

### 模型不输出新字段

把生成说明写得更明确，并确认当前模板字段列表中已经保存了这个字段。

### 数值不显示 +0

标准状态模板要求无变化输出 `+0`。新版前端也会把空变化规范化为 `+0`。

## 8. 记住边界

```text
模板决定输出什么内容。
美化包决定这些内容怎么显示。
```
