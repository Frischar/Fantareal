About

Fantareal 的开端

一个让 AI 聊天与做人设更加轻便简易的软件。

很多项目的诞生，并不是因为一开始就想创造什么宏大的东西。

有时候，她只是源于一个很朴素的愿望。

当渐渐庞杂的结构让体验变得迟缓，当层层叠叠的规则把本该属于创作者的自由困在狭窄的框架里。那些原本只是为了整理设定、保存记忆、延续故事而存在的功能，似乎也慢慢变成了新的负担。

于是，有人开始动手。试着一点点拆开、理顺、重写，让那些本可以更轻、更好用的东西，真正回到使用者手里。

这个人叫 绪亓。

走到现在，早已不是我一个人的努力了。苏打 作为核心开发者，几乎与我一同撑起了这个项目。还有那些不分昼夜帮忙排查问题的伙伴、给出反馈的测试者——没有他们，这间屋子搭不起来。

在那之前，许多尝试曾经发生过。有人出于善意分享经验，有人试着让更多人少走弯路，也有人真心希望大家交流的角落能变得更好。只是有时候，认真付出的心血，未必总能被妥善安放；曾经留下的痕迹，也可能会在不经意的风波中无奈散去。

既然有些门关上了。

那我们就开始搭另一间屋子。

这间屋子，后来被命名为 Fantareal。

Features

功能亮点

对于初学者

我们的「轮椅模式」可以让他们更加轻松地实现自己的想法：向 AI 告知自己的想法，AI 整理出实现你想法的文字，将其自动放到该放的位置上。creator-notes 等角色卡内容应用内翻译为中文，让外语过敏者也可以舒适制作人设。

对于玩家

Fantareal 更加小巧精致，较小内存及界面更加简洁。

喜欢的图片自己添加为背景图，喜欢的音乐自己添加为背景音乐：这些都是可以自己设置的，不会被官方限制为某个角色，自己喜欢的就是最好的。

对于中度人设爱好者

他们可能想要制作更加复杂的人设，但其他 AI 软件有的学习成本高且繁琐，有的下限低。Fantareal 没有输出的限制，格式不会塞给你多余没用的内容消耗 API。

世界书（检索到关键词，将词条内容作为一种补充融入到人设聊天中）使 API 消耗更小，从此不再担忧 API 消耗昂贵。

轮椅模式（向 AI 告知自己的想法，AI 整理出实现你想法的文字，将其自动放到该放的位置上）节省时间，一键替换让你解放双手。

演出工坊（播放开幕效果与音乐等）加强人设的沉浸感，让你内心的想法更加好地展现。
